## 任务目标

基于项目现有技术栈（TailwindCSS + Three.js）开发/修改页面，实现科技感炫酷视觉效果；代码精简、最小化Token消耗，**每次修改必须完成单元测试**。

## 触发句式

- 用Three.js+Tailwind做科技感页面
- 优化这个页面的3D动效
- 修改界面并做单元测试
- 开发炫酷科技风UI，省Token
- 只改代码，加单元测试

## 执行流程（必须严格按顺序）

1. **技术栈约束**
   - 仅使用项目已集成的 TailwindCSS、Three.js
   - 不新增依赖、不引入新框架、不修改项目配置
   - 保持现有工程结构、打包流程不变

2. **代码与Token优化**
   - 代码极简、无冗余、无多余注释
   - 单文件优先，结构清晰
   - 严格控制输出长度，节省Token

3. **视觉与动效要求**
   - 科技感、未来风、赛博朋克风格
   - 支持3D粒子、动态背景、霓虹渐变、磨砂玻璃、流光、悬浮呼吸动画
   - 交互流畅、轻量化、高性能
   - 全屏响应式布局

4. **单元测试强制要求**
   - 仅针对本次修改/新增文件编写测试
   - 覆盖：DOM渲染、样式加载、3D场景初始化、动画启动
   - 检查语法、无报错、无控制台异常
   - 自动运行验证，确保页面可正常加载

5. **输出与交付**
   - 输出可直接运行的完整代码
   - 不破坏原有项目逻辑
   - 附带单元测试代码+运行命令
   - 确保无报错、可直接提交

## 输出格式

1. 本次修改文件清单
2. 完整可运行代码
3. 单元测试代码
4. 运行命令 + 验证结果
