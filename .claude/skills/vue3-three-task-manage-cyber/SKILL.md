===== 一、视觉强制约束（必须严格执行）

1. 底色：分为暗黑主题和明亮主题；
2. 主配色：电光青 #00E5FF、极光紫 #9D5CFF、荧光橙 #FF7D00、银白 #E8F0FF，所有光效、高亮、选中态仅使用这四色渐变/纯色；
3. UI材质：2D层统一使用玻璃拟态（半透明+模糊+细描边），3D模型优先使用玻璃材质、自发光材质、金属哑光材质；
4. 字体：无衬线粗体科技字体，文字层级分明，次要文字低透明度，禁止花哨艺术字体。

===== 二、Three.js 3D场景强制规则（核心）

1. 全屏渲染Three.js场景，作为页面底层背景，画布铺满全屏，层级z-index最低；
2. 场景标配：星空粒子系统 + 动态网格地面 + 主辅双光源 + 环境光，永久低缓动画，帧率稳定60帧；
3. 所有任务卡片、分类标签、统计图表均使用Three.js几何体建模，不使用纯DOM列表；
4. 3D模型基础动画：所有悬浮模型默认开启微小上下浮动 + 极慢自转，幅度柔和，无眩晕感；
5. 相机：支持平滑缓动切换视角，页面切换、模块选中时执行相机转场动画。

===== 三、动画&交互强制规则（GSAP实现，所有交互必须带动效）

1. 智能Hover行为：3D卡片放大1.1倍、抬升、边缘发光；2D元素光晕扩散、弹性形变；
2. 点击选中：模型弹起+小幅旋转，生成环形粒子光圈特效；
3. 新增任务：光带放射 + 卡片弹出动画；完成任务：卡片坠落+粒子爆炸消散；删除：扭曲收缩+烟雾粒子；
4. 拖拽交互：拖拽物体生成光轨拖影，放置后吸附回弹，目标区域触发范围光效；
5. 页面/模块切换：强制使用相机平滑转场 + 全屏光幕过渡，禁止生硬跳转；
6. 进度/数据：3D流光进度条、数字滚动跳动+脉冲光点；
7. 所有动画缓动函数统一使用弹性/缓出曲线，速度适中，禁止生硬线性动画。

===== 四、代码&架构约束

1. Vue3 全部使用 <script setup> 语法，组合式API拆分功能模块；
2. Three.js 单独封装全局场景类、粒子类、模型类、动画类，解耦业务与3D渲染；
3. 动效统一使用GSAP管理，禁止原生setTimeout/setInterval做复杂动画；
4. 响应式适配：3D场景、UI布局自动适配不同屏幕尺寸，移动端保留核心动效；
5. 性能优化：粒子数量、模型面数合理控制，闲置动画降频，保证低配置设备流畅运行；

===== 五、禁止项（严格禁止）

1. 禁止传统扁平化老旧列表样式；
2. 禁止无动效、无反馈的生硬交互；
3. 禁止高饱和刺眼大面积色块；
4. 禁止Three.js与DOM层级混乱、遮挡错乱；
5. 禁止冗余无效代码，模块拆分清晰，注释规范。

开发要求：每一部分代码、页面、组件、动效均严格遵循以上规则，输出完整可运行代码，附带功能说明与动效说明。
